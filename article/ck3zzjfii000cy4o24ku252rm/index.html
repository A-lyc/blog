  • web

主要讨论以下属性

  • clientHeight / clientWidth
  • offsetHeight / offsetWidth
  • scrollHeight / scrollWidth

包罗万象

看懂了就啥都明白了

clientHeight / clientWidth

包含 padding，不包含 border、margin、滚动条尺寸

offsetHeight / offsetWidth

包含 padding、border、滚动条尺寸，不包含 margin

scrollHeight / scrollWidth

此属性一般用于有滚动条的元素
返回滚动内容的总尺寸
